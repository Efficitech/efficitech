SchoolOrganized

Bookmarks Organizer

View Project

We created SchoolOrganized because there is a problem with chrome bookmarks. When you've visited a lot of sites and have them bookmarked, the bookmarks bar can get cluttered, making it difficult to access important sites quickly. So we created SchoolOrganized, we've added new features like searching bookmarks, customization, built in calculator and notes and better ui. This tool was mainly targeted at students but it is still useful even if you are not a student.
